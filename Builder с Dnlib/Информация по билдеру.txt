Билд - это ваш проект который вы хотите использовать для замены данных через билдер
Билдер - это конструктор который изменяет ваш исходный код файла на пользовательские данные.

Пример:

Build - есть текст [MYTEST]
В Builder'e пользователь вводит текст и строка [MYTEST] заменяется на Вашу строку. и.т.д
В конечном выходном файле будет изменённый файл с Вашими данными.


Билд закидываем в ресурсы билдера.
